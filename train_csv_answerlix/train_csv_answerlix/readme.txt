#Instructions file to train AnswerLIX :

1)Insert the new csv,test questions,train(for classification --> question and target column),train (for NER --> only questions) in the input_files folder. Insert the key files for train questions in the outputs folder.
2)Run the main.py
3)The model is saved to disk as MODEL_FINAL.
4)The pkl files will be saved in pkl_files folder.Copy these to the AnswerLIX input_model_pkl_files folder.
5)To check the accuracy run the accuracy.py with inputs as key file and answers generated by model.
